## Vancouver Trees Explorer

This R Shiny App **Vancouver Trees Explorer** can help you to take a look at trees genus and distribution in different locations.

Check out the website [here](https://circle117.shinyapps.io/assignment-b3-joy-yu/)!

There are three main features of this app:

* FEATURE 1: A tree genus location map showing the distribution of trees and their genus that updates with the input neighbourhood, street name, genus or exact longitude and latitude location
* FEATURE 2: Overview tables by neighbourhoods or streets showing the total number of trees and number of genera in one area
* FEATURE 3: A button for the users to download the filtered data as a .csv file
